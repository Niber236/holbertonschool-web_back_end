# NoSQL

Ce projet introduit les concepts des bases de données NoSQL, en se concentrant sur MongoDB et son utilisation via des scripts shell et Python (PyMongo).

## Objectifs d'apprentissage
* Comprendre ce que signifie NoSQL et la différence avec SQL.
* Comprendre le concept ACID et le stockage orienté document.
* Connaître les avantages d'une base de données NoSQL.
* Savoir interroger, insérer, mettre à jour et supprimer des informations dans MongoDB.
* Utiliser PyMongo pour interagir avec MongoDB depuis Python.

## Environnement
* OS: Ubuntu 20.04 LTS / 22.04 LTS
* Base de données: MongoDB 4.4
* Langage: Python 3.9
* Librairie: PyMongo 4.8.0
